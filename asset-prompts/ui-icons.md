# UI Icons (SVG + PNG)

All icons should be **readable at 24×24**, with a strong silhouette and minimal interior clutter.

Each icon includes:
- Prompt (SVG): generates two SVG variants (Night UI + Tabletop Daylight)
- Prompt (PNG): generates two transparent PNG variants (Night UI + Tabletop Daylight)

## Navigation / utility
