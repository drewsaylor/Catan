# Structures & Badges

Prompts for build pieces (road/settlement/city), plus achievement and event badges.

## Build pieces (icon + tabletop piece variants)

### piece-road — Road piece

Use: Build UI + board highlights

Output: 1024×1024 transparent PNG (generate two sub-variants per main variant)

Prompt (PNG):
```text
Make two variants: (A) Night UI and (B) Tabletop Daylight.
Must be original; do not copy any existing board game’s trademarked logos/characters/layouts.
No watermarks, signatures, or model marks.
No text unless explicitly requested.
Keep shapes readable at small sizes; strong silhouette; minimal clutter.

Output: 1024x1024 transparent PNG.

Create TWO versions for each variant:
1) Flat UI icon version: simple road plank/beam segment, minimal.
2) Tabletop piece version: small 3D wooden road piece (meeple-like), subtle shading.

Composition: centered, same silhouette between versions.
Variant A (Night UI): slightly darker wood with cool rim light.
Variant B (Tabletop Daylight): warm wood with sunlit highlight.
```
