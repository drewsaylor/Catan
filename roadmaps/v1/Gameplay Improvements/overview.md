# Gameplay Improvements — Overview

## Goals

- Support **3–6 players** with good pacing and rules safety.
- Add **house rules** without breaking “Jackbox clarity first.”
- Add lightweight **social features** that don’t require moderation.

## Dependencies

- UI Improvements Phase 1/2: 6-player layout scaling.
- Reliability Improvements Phase 2: action idempotency (prevents double-build, double-end-turn, etc.).

## Phases

- Phase 1 (P0): 3–6 players + piece limits
- Phase 2 (P1): house rules & room settings
- Phase 3 (P2): social + trade enhancements
